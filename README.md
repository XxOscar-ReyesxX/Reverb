# Reverb
Implementación de un efecto de reverberación digital mediante la convolución de una señal de audio con una respuesta al impulso, aplicando conceptos de procesamiento digital de señales.

El archivo audio_reverb.wav contiene el audio resultante depsues de haberse aplicado el efecto de reverberación
